
# nodejsmysql

Connecting nodejs to mysql and performing queries using html forms 

## Run Locally

Clone the project

```bash
  git clone https://github.com/Siddharth189/nodejsmysql
```

Go to the project directory

```bash
  cd nodejsmysql
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm server.js
```
